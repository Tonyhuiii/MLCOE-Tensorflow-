# MLCOE-Tensorflow-
Tensorflow implementation of paper:  
Diffusion-based Time Series Imputation and Forecasting with Structured State Space Models (https://arxiv.org/abs/2208.09399).  
## Environment
● Ubuntu 20.04  
● NVIDIA RTX 3090
## Prerequisites
● Python 3.9.13  
● [Tensorflow](https://pytorch.org/) 2.10.0  
● cudatoolkit 11.2.2  
● cudnn 8.1.0
